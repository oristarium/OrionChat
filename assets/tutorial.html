Chat Tools Tutorial

This guide will help you set up Chat Tools in OBS for your streaming needs. There are three main components to set up:

1. Control Panel Setup

The control panel is where you manage your chat connections and settings.

  1. In OBS, go to View → Docks → Custom Browser Docks
  2. Add a new dock with:
    • Dock Name: Chat Tools Control
    • URL: http://localhost:7777
  3. Click "Apply" to add the dock to your OBS workspace

2. Avatar TTS Setup

The TTS avatar will appear and speak chat messages on your stream.

  1. In OBS, add a new Browser source to your scene
  2. Configure it with:
    • URL: http://localhost:7777/tts
    • Width: 300 (or as needed)
    • Height: 300 (or as needed)
  3. Position the browser source where you want the avatar to appear

3. Chat Display Setup

This will show highlighted chat messages on your stream.

  1. Add another Browser source to your scene
  2. Configure it with:
    • URL: http://localhost:7777/display
    • Width: 400 (or as needed)
    • Height: 600 (or as needed)
  3. Position the chat display where you want messages to appear
Tip: Make sure Chat Tools is running before adding these sources to OBS. The application needs to be running for the sources to work properly.

Usage Tips

  • Use the control panel to connect to your streaming platform
  • You can customize the avatar by clicking the "Avatar Settings" button
  • Select your preferred TTS language from the dropdown menu
  • The chat display will automatically show messages as they come in
