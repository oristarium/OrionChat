OrionChat Tutorial

This guide will help you set up OrionChat for your streaming needs.

1. Control Panel Setup in OBS

The control panel is where you manage all your OrionChat settings.

  1. In OBS Studio, go to "View" → "Docks" → "Custom Browser Docks..."
  2. In the popup window, add:
    • Dock Name: OrionChat Control
    • URL: http://localhost:7777
  3. Click "Apply" then "Close"
  4. The Control Panel will appear as a dockable window in OBS
  5. You can now arrange it alongside your other OBS panels

2. Avatar Setup

2.1 Browse Available Avatars

  1. Open http://localhost:7777/avatar in your browser
  2. View all available avatars and their configurations
  3. Each avatar card shows:
    • Avatar ID (needed for setup)
    • Preview of idle and talking states
    • Available voice options

2.2 Adding Avatar Sources in OBS

  1. In OBS Studio, right-click in the "Sources" panel
  2. Select "Add" → "Browser"
  3. Create new and name it (example: "Avatar 1")
  4. Set URL to http://localhost:7777/avatar/{avatar_id}
    Replace {avatar_id} with the actual ID from the avatar index
  5. Set dimensions:
    • Width: 1080
    • Height: 1080
  6. Repeat for each avatar you want to display
Important:
  • Each avatar needs its own browser source
  • Avatars will coordinate automatically to prevent speech overlap
  • Messages are distributed among available avatars

3. Using with VTube Studio

  1. Open VTube Studio
  2. Click the "+" button in the bottom right to add a new item
  3. Select "Web Item"
  4. For each avatar:
    • URL: http://localhost:7777/avatar/{avatar_id}
    • Recommended size: 1080x1080
    • Position as needed in your scene

For more detailed information about Web Items in VTube Studio, please refer to the official VTube Studio Wiki.

Tips for Content Creators

  • Make sure OrionChat is running before starting OBS or VTube Studio
  • Test your setup before going live
  • The Control Panel can be accessed from any browser at http://localhost:7777
  • Customize your avatar and chat display settings through the Control Panel
  • For best results, keep the application running in the background while streaming

Troubleshooting

  • If you can't see the avatars, make sure OrionChat is running
  • Try refreshing your browser sources in OBS or web items in VTube Studio
  • Check if your antivirus is blocking the application
  • Make sure you're using the correct avatar IDs in your URLs
  • If nothing works, restart OrionChat and your streaming software
